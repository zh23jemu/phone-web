# 仿真电话 App (uniapp)

## 项目简介

本项目是一个使用 uniapp 开发的仿真电话和短信功能的应用程序。旨在模拟原生手机的拨号、通话、通话记录、来电提醒和短信收发界面及基本交互，不涉及真实的通讯功能。

## 主要功能

### 电话相关
- **拨号盘**:
    - 数字 (0-9, \*, #) 按键，支持按键音效。
    - 输入号码实时显示。
    - 删除按钮。
    - 拨号按钮启动模拟呼叫。
- **模拟通话界面**:
    - 显示当前拨打/来电号码。
    - 免提、静音按钮状态切换。
    - 红色挂断按钮。
    - 通话计时器 (模拟接通后)。
    - 模拟接通/未接通/呼叫中状态。
- **通话记录**:
    - 列表展示历史模拟通话。
    - 显示号码、通话时间、(未来可加入)通话类型。
    - 点击记录可快速回拨。
    - 显示号码备注/标记 (例如："快递", "会议提醒")。
- **模拟来电**:
    - 可由后端触发，在前端弹出模拟来电界面。
    - 显示来电号码及预设备注。
    - 提供"接听"和"挂断"选项。

### 短信相关
- **短信列表**:
    - 展示模拟收到的短信概览。
- **短信详情**:
    - 查看单条短信的完整内容。
- **模拟接收短信**:
    - 可由后端触发，在前端显示新短信通知及内容。

### 后端联动 (预期)
- **记录管理**: 保存和查询模拟通话记录。
- **号码标记**: 允许用户为特定号码添加自定义标记或分类，并在来电或拨号时显示。
- **远程控制**: API接口控制前端模拟来电和接收短信。

## 快速开始

### 环境准备
- 安装 HBuilderX：前往 [DCloud官网](https://www.dcloud.io/hbuilderx.html) 下载并安装。
- (可选) 安装 Node.js：如果需要运行一些npm脚本或者使用相关CLI。

### 运行项目
1.  克隆或下载本仓库到本地。
2.  使用 HBuilderX 打开项目。
3.  配置 `manifest.json` 中的 AppID (如果需要发布到小程序或App平台)。
4.  在 HBuilderX 中选择运行的目标平台：
    *   **运行到浏览器**: 选择 Chrome/Firefox/Edge 等。
    *   **运行到手机或模拟器**: 需要连接手机或启动Android/iOS模拟器。
    *   **运行到小程序模拟器**: 需要配置相应的小程序开发者工具路径。

### 构建项目
- 在 HBuilderX 中选择"发行"，然后根据需要选择目标平台进行打包。
